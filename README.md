# 🎨 Pixel Art Maker

A beautiful, interactive pixel art creation tool built with HTML, CSS, and JavaScript. Create stunning pixel art by coloring individual squares on a customizable grid!

## 🎮 How to Use

1. **Choose Grid Size**: Select from 16x16, 32x32, or 64x64 pixels
2. **Pick a Color**: Use the color picker to select your desired color
3. **Start Drawing**: 
   - Click on pixels to color them
   - Click and drag to draw continuously
4. **Use the Eraser**: Click the eraser button to remove colors
5. **Clear Canvas**: Use the clear button to start over
6. **Save Your Art**: Click the save button to download your artwork

## 🛠️ Technical Details

### Built With
- **HTML5**: Semantic structure and accessibility
- **CSS3**: Modern styling with gradients, animations, and responsive design
- **Vanilla JavaScript**: ES6+ classes and modern DOM manipulation
- **Canvas API**: For high-quality image export


## 🚀 Getting Started

1. **Clone or Download** the project files
2. **Open `index.html`** in your web browser
3. **Start Creating** your pixel art masterpiece!

No build process or dependencies required - it's ready to use immediately!

## 🎨 Tips for Great Pixel Art

- **Start Simple**: Begin with basic shapes and outlines
- **Use Limited Colors**: Pixel art often looks better with fewer colors
- **Plan Your Design**: Sketch your idea before starting
- **Use the Grid**: The grid helps you create precise shapes
- **Experiment**: Try different grid sizes for different effects
